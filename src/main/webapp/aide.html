Conseils pratiques

Comment créer mon compte passager sur Coplaning?
Créer un compte ne prend que quelques secondes et est nécessaire pour faire une demande de réservation auprès des pilotes. Il vous suffit simplement de vous inscrire en cliquant sur le bouton "inscription", puis compléter les informations demandées.
Comment faire une demande de réservation pour un vol ?
Pour réserver un vol il fautse connecter avec ses identifiants, ou créer un compte si vous n'en avez pas. Vous devez ensuite rechercher un vol en utilisant le fonction de recherche du site, puis cliquer sur le bouton réserver lorsque vous avez trouvé votre vol.
Comment supprimer mon compte Coplaning ?
Une fois connecté, rendez-vous sur la page de votre profil Coplaning. Selectionné ensuite la rubrique "Paramètres de compte" puis cliquer sur le bouton "Supprimer".

ATTENTION : vous ne pourrez plus récupérer votre compte une fois celui-ci supprimé (vous pourrez en revanche en créer un autre).
Comment s'inscrire en tant que pilote sur le site de Coplaning
Créer un compte en tant que pilote ne prend que quelques secondes et est nécessaire pour être en mesure de proposer des vols sur le site de Coplaning. Il vous suffit simplement de vous inscrire en cliquant sur le bouton "Vous êtes pilote?", puis compléter les informations demandées.

REMARQUE : une attestion de pilote (PPL, CPL, etc...) vous sera demandée pour justifié de votre statut de pilote.
Comment annuler une réservation pour vol ?
Vous pouvez annuler une réservation de vol 48h avant l'heure de début de celui-ci. Pour cela il suffit de vous rendre sur la page de votre profil dans la rubrique "Réservations" puis de cliquer sur le bouton "Annuler".
Comment effectuer une recherche de vol ?
Pour cela il vous suffit de rentrer les informations nécessaires dans la barre de recherche de la page d'accueil puis de cliquer sur le bouton "Rechercher".
